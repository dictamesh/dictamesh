# DictaMesh Governance Package

Access control, PII tracking, and compliance features for the DictaMesh framework.

## Features

- Access control policy engine
- PII detection and tracking
- Audit logging framework
- Data classification
- Compliance rule engine (GDPR, CCPA)
- Data masking utilities
